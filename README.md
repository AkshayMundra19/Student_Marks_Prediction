# Student_Marks_Prediction
A beginner-friendly Machine Learning project using Linear Regression to predict student marks based on study hours, with data visualization and performance evaluation.

# Study Hours vs Marks Prediction

This project demonstrates a simple **Linear Regression** model that predicts student marks based on study hours.

## Project Overview
- Input: Study Hours
- Output: Predicted Marks
- Algorithm: Linear Regression
- Language: Python

## Dataset
The dataset is manually created with the following columns:
- Hours: Number of hours studied
- Marks: Marks obtained

## Libraries Used
- pandas
- matplotlib
- scikit-learn
